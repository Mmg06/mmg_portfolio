Project 3: Bitcoin in E-Sports

The dataset used for this E-Sports analysis includes 4.5 million rows across 25 columns, tracking detailed player and match data from PUBG games. Analytical techniques like correlation analysis and machine learning, specifically XGBoost, were employed to predict player rankings with a high accuracy, indicated by a mean squared error (MSE) of 0.3. Future work aims to refine the scoring system by emphasizing the kill-to-death (KD) ratio over survival, potentially incorporating Bitcoin rewards to better reflect player skills and make the game more competitive and financially rewarding.

Project 4: Breast Cancer Diagnosis

This project utilized the Breast Cancer Wisconsin (Diagnostic) dataset, which contains 596 rows and 32 columns, to classify tumors as benign or malignant based on their geometrical features. These features were extracted from digitized images of fine needle aspirates (FNA) of breast masses and describe the cell nuclei's characteristics. Key features include tumor radius, texture, perimeter, area, smoothness, compactness, concavity, symmetry, and fractal dimension. For each image, the mean, standard error, and largest values ("worst") of these features were computed, resulting in 30 key metrics. The analysis applied the k-nearest neighbors (k-NN) algorithm to classify tumors with high accuracy, demonstrating the model's effectiveness in distinguishing between benign and malignant cases.

Project 5: Statistical Analysis of Internet Usage in the UK.

This project analyzed internet usage trends in the UK across various demographics, including age, gender, ethnicity, disability, and economic groups. The objective was to assess internet usage patterns over time to guide investment decisions and identify target groups for maximizing profits. Data was processed and analyzed using statistical techniques and machine learning models like Linear Regression and Decision Trees, which helped predict usage trends and highlight key demographic factors influencing internet adoption. The insights provided a data-driven approach for strategic market targeting, supporting informed decision-making for potential investments.

Project 6: Project Owl Image Processing

The project aimed to develop image processing algorithms to enhance fundus images for detecting diabetic retinopathy symptoms. Using MATLAB, the team removed image artifacts (like reflections) and highlighted key features such as hard exudates and blood vessel proliferation. SURF and SIFT were employed for reflection removal, and the Kirsch operator was used for vessel detection. The green channel was processed for better contrast, and exudate areas were quantified through binary image analysis. The next step involves applying machine learning for disease detection, but this exceeds the current scope. Challenges included overlapping features and edge detection.

Project 7: Sales Management using PowerBI

This project involved building a dynamic sales dashboard using SQL and Power BI, based on a business request to improve internet sales reporting. The dataset was extracted from a Microsoft sample database and cleansed using SQL. Key tables such as DIM_Customer, DIM_Product, and FACT_InternetSales were transformed to meet business needs. A data model was created in Power BI, linking the sales data with budget comparisons, and enabling filtering by customer, product, and time period. The final dashboard provided a visual overview of sales trends, helping sales managers and representatives track performance, compare against budget, and identify key sales opportunities.
